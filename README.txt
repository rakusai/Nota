■紹介
コミュニティやプロジェクトで共有できる参加型コミュニケーショ
ン・プラットフォームの登場です。NOTA（ノータ）は、多人数で同
時に（リアルタイムに）一つのページを編集できます。もちろん、
デザインは自由自在。ウェブブラウザ上の好きな位置に、文字を書
いたり、絵や図を描いたり、写真を貼り付けたりできます。自由で
多彩な表現を駆使して、今までにないコミュニケーションを体感し
てください。

■NOTAネットワークついて
NOTAは、プログラマーだけでなく、一般市民も参加する「NOTAネッ
トワーク」というプロジェクトを通じて、配布、開発、サポート、
事例紹介などの活動を行ってきます。最新の記事はいつでも本サイ
トに投稿されます。興味を持たれた方は、気軽に私たちにご連絡く
ださい。
